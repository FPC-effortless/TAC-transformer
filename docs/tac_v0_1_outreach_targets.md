# TAC v0.1 Outreach Targets

Goal: start a focused feedback and funding loop for TAC v0.1 without
overclaiming. This list uses public pages and should be treated as a starting
queue, not a scraped contact database.

Current repository branch:

https://github.com/FPC-effortless/TAC-transformer/tree/tac-v0.1-public

Current caveat: if this link returns 404 to a logged-out browser, the repository
is still private or otherwise not publicly visible. Make the repository public
before sending broad outreach.

## Outreach Order

1. Fix public repo visibility.
2. Record demo.
3. Send 5 researcher-feedback messages.
4. Send 3 investor-feedback messages.
5. Apply to one accelerator or grant path only after the first feedback replies.

## Minimum Package To Send

- GitHub branch link.
- Kaggle validation link.
- Architecture diagram.
- Technical report.
- Limitations.
- Demo video.
- One paragraph asking for critique, not money.
